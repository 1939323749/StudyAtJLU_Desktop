# 学在吉大视频下载器/H5播放器

基于[学在吉大视频下载器/H5播放器](https://github.com/RikaCelery/StudyAtJlu)

## 功能

+ 视频筛选(按课程，按学期)
+ 视频下载(按课程，按学期)
+ 自动分类(保存文件夹/课程名/日期/视频类型.MP4)
+ 跳过已下载文件(~~没得写了，凑字数~~)
+ 播放器(同步播放，可拖动进度条)
